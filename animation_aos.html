以下動畫套用AOS效果，在JS加入
AOS.init({
duration: 800,
anchorPlacement: 'top-center'})
設定動畫時間0.8秒、碰觸到頂端才觸發動畫

fade-right

由下方淡入+照片由左翻轉

fade-down、flip-left

外層div加入fade-down class、照片加入flip-left class

由上方淡入+照片放大

fade-up、zoom-in

外層div加入fade-up class、照片加入zoom-in class

由右側滑入+照片另外設置動畫完成秒數

slide-right

外層div及照片皆加入slide-right class
